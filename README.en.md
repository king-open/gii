## 📦 Installation

```bash
# npm
npm install gii

# yarn
yarn add gii

# pnpm
pnpm add gii
```

## 🔨 Usage

```tsx
import { Button } from 'gii';

const App = () => (
  <Button type="primary">
    Hello GII!
  </Button>
);
```

## 🎯 Development

```bash
# Install dependencies
$ npm install

# Start development server
$ npm start

# Build component library
$ npm run build

# Build documentation
$ npm run docs:build
```
